Class Extensible.calendar.template.Month

Package:Extensible.calendar.template
Defined In:Month.js
Class:Month
Extends:Object

This is the template used to render the MonthView. Internally this class defers to an instance of Ext.calerndar.BoxLayoutTemplate to handle the inner layout rendering and adds containing elements around that to form the month view.

This template is automatically bound to the underlying event store by the calendar components and expects records of type Extensible.calendar.data.EventModel.
