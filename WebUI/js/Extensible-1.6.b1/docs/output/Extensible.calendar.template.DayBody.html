Class Extensible.calendar.template.DayBody

Package:Extensible.calendar.template
Defined In:DayBody.js
Class:DayBody
Extends:Object

This is the template used to render the scrolling body container used in DayView and WeekView. This template is automatically bound to the underlying event store by the calendar components and expects records of type Extensible.calendar.data.EventModel.

Note that this template would not normally be used directly. Instead you would use the Extensible.calendar.view.DayTemplate that internally creates an instance of this template along with a Extensible.calendar.DayHeaderTemplate.
