Class Extensible.calendar.util.ColorPicker

Package:Extensible.calendar.util
Defined In:ColorPicker.js
Class:ColorPicker
Extends:Object
Simple color picker class for choosing colors specifically for calendars. This is a lightly modified version of the default Ext color picker that is based on calendar ids rather than hex color codes so that the colors can be easily modified via CSS and automatically applied to calendars. The specific colors used by default are also chosen to provide good color contrast when displayed in calendars.
